## Use the Javascript Console
You can use `typeof` to check the `data structure`, or type, of a variable.

Note that in JavaScript, arrays are technically a type of object.

```js
console.log(typeof(""));

console.log(typeof(0));

console.log(typeof([]));

console.log(typeof({}));
```
